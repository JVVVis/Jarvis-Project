# Jvis - AI Mobile Assistant

Jvis is a native mobile AI assistant application designed to help users with day-to-day challenges and answer questions through voice and text interaction.

## Project Structure

- **iOS/** - Native Swift iOS application
- **Android/** - Native Kotlin Android application
- **backend/** - Node.js/Express API backend for AI integration
- **docs/** - Documentation and architecture guides

## Features

- 🎤 Voice input recognition
- 💬 Real-time conversational AI
- 🔊 Natural speech output
- 📝 Conversation history
- ⚙️ Customizable settings
- 🔐 Secure API integration
